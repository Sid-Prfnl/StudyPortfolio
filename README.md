## Featured Project: Dirk Nowitzki Performance Analysis

### Objective
Analyze and model Dirk Nowitzki’s career performance to identify efficiency trends and scoring drivers.

### Dataset
Per-game career statistics (NBA historical dataset).

### Methods
- Data cleaning and validation framework
- Advanced efficiency metrics (TS%)
- SQL-based aggregation
- Machine learning regression models
- Interactive Streamlit dashboard

### Key Insights
- Peak scoring efficiency occurred in mid-career seasons.
- Field goal attempts and minutes were strongest predictors of points.
- Efficiency remained stable despite late-career decline in volume.
